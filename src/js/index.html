3. Function showModal(type,title,msg,flag)

Param type called to style modal window. In basic script 4 default style: 0 - error msg, 1 - Success msg, 2- Info msg, 3 - Warning msg
type - integer param . Is a required parameter.

Param title called to header title modal window.title - string param Is a required parameter, take only text string, not HTML.

Param msg called to output message. You can display here any code HTML. msg - string param

Param flag - optional. if set 1, You can output GLOBAL variables, who named must be indicated in param msg. flag - integer param . is an optional parameter
